# Exploratory Data Analysis — Books Dataset
## Task 2

## Project Overview
Performed EDA on a self-scraped dataset of 1000 
books from books.toscrape.com to uncover patterns 
in book pricing and ratings.

## Business Questions Answered
- What is the average price of books?
- How are ratings distributed?
- Does price affect rating?
- Which books are most expensive?
- Which books have highest ratings?

## Dataset
- Source: Self scraped from books.toscrape.com
- Rows: 1000 books
- Columns: Title, Price, Rating

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
1. Average book price is around £35
2. Ratings are evenly distributed 1-5 stars
3. No strong correlation between price and rating
4. Most books priced between £10-£60
5. Five star books exist across all price ranges

## Visualizations
- Price Distribution (Histogram)
- Rating Distribution (Count Plot)
- Price by Rating (Box Plot)
